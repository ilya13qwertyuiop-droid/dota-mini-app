# assets

Статика мини-аппа, отдаётся nginx из корня проекта по относительному пути.

## tinker-meme.png
Картинка мем-оверлея «Выше / Ниже». Выезжает на весь экран, когда в режиме
«Кто чаще умирает» раскрываются аномальные 46 смертей Тинкера — дань
комьюнити-мему. Положи сюда файл с именем ровно `tinker-meme.png`.

- Формат: PNG (можно с прозрачностью) или JPG — тогда поправь путь в
  [index.html](../index.html) (`#mghl-meme-img src`).
- Рекомендуемый размер: до ~1080px по большей стороне, оптимизированный вес.
- Если файла нет — оверлей просто не сработает (onerror помечает его broken),
  игра продолжает работать как обычно.
